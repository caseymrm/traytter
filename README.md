# traytter

> **Deprecated — no longer works.** Traytter scraped public twitter.com profile pages, which X locked behind a login wall in 2023. The HTML structure it parsed is also long gone, so the app will not fetch tweets. Repo is kept for historical reference. If you want a similar menu-bar feed today, point an RSS-aware client at a self-hosted [Nitter](https://github.com/zedeus/nitter) instance.

Minimalist Twitter client for the OSX menu bar

## Install

* [Download the app](https://github.com/caseymrm/traytter/releases/download/v0.1/Traytter.zip)
* Unzip it
* Put it in Applications
* Run it
* To run every time you login, select "start at login" from the menu bar
